Alma Dulce: Sistema Web de Pedidos para Pastelería

Alma Dulce es una aplicación web full-stack diseñada para gestionar el ciclo completo de pedidos de una pastelería artesanal. La plataforma permite a los usuarios navegar por los productos, personalizar sus pasteles y realizar pedidos, mientras que los administradores pueden gestionar el inventario, los precios y el estado de cada solicitud en tiempo real.

Toda la información de productos (sabores, tamaños, decoraciones y precios) se gestiona centralmente a través de una base de datos MySQL.

Características Principales

Funcionalidades de Usuario

Registro y Login Seguro: Los usuarios pueden crear cuentas nuevas para acceder a la plataforma.

Creación de Pedidos: Los usuarios registrados pueden generar nuevos pedidos, especificando sabor, tamaño, decoraciones y un mensaje personalizado.

Seguimiento de Pedidos: Cada usuario puede ver el historial de sus pedidos realizados y verificar su estado (Pendiente, Procesando, Completado, etc.).

Funcionalidades de Administración

Panel de Control Centralizado: Acceso exclusivo a una vista detallada de todos los pedidos de todos los usuarios.

Gestión de Estados: Los administradores pueden cambiar el estado de cualquier pedido para reflejar el progreso de la producción.

Edición Completa: Capacidad para modificar los detalles de un pedido existente (sabor, tamaño, decoración) y el precio total, el cual se recalcula automáticamente según las opciones seleccionadas.

Eliminación de Pedidos: Control total para eliminar pedidos.
